# WILL DTF - Loja de Camisetas Personalizadas

Uma aplicação web moderna para criar e vender camisetas personalizadas com estampas DTF (Direct-to-Film).

## 🚀 Tecnologias

- **Next.js 14** - Framework React com SSR
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização responsiva
- **ShadCN UI** - Componentes reutilizáveis
- **Framer Motion** - Animações fluidas

## 📋 Requisitos

- Node.js 18+ instalado
- npm ou yarn

## 🛠️ Instalação e Configuração

### 1. Instalar dependências

```bash
npm install
# ou
yarn install
```

### 2. Executar servidor de desenvolvimento

```bash
npm run dev
# ou
yarn dev
```

A aplicação estará disponível em: **http://localhost:3000**

### 3. Build para produção

```bash
npm run build
npm run start
```

## 📝 Funcionalidades

✅ Criação de camisetas personalizadas
✅ Seleção de cores e tamanhos
✅ Carrinho de compras funcional
✅ Remoção de itens do carrinho
✅ Cálculo automático de total
✅ Interface responsiva
✅ Animações com Framer Motion
✅ Design moderno e profissional

## 🎨 Componentes Principais

- **WillDTFStore** - Componente principal da loja
- **Card** - Componente de cartão reutilizável
- **Button** - Botão com variantes
- **Input** - Campo de entrada estilizado
